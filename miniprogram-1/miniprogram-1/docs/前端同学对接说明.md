# 前端同学对接说明（Language for Life）

> 后端/AI 负责人已完成接口与 Alpha 演示页。你按墨刀重做 UI 时，**请优先改 `pages/*/wxml、wxss`，不要改 `utils/api.js` 和 `utils/coze.js` 除非联调需要。**

## 一、你怎么拿到项目

### 方式 A：Git（推荐）

1. 负责人把代码推到 GitHub/Gitee 仓库  
2. 你 `git clone` 到本地  
3. 用微信开发者工具打开项目根目录（含 `project.config.json` 的文件夹）

### 方式 B：压缩包

负责人打包整个 `miniprogram-1` 文件夹发你（微信/网盘），解压后用开发者工具导入。

### 方式 C：已是小程序协作者

1. 登录 [微信公众平台](https://mp.weixin.qq.com/)  
2. **开发 → 开发管理 → 成员管理** 里应有你的账号  
3. 本机安装「微信开发者工具」，用**你的微信**扫码登录  
4. **导入项目** → 选负责人给的目录，AppID 选团队小程序（与负责人一致）

## 三、数据接口怎么用（核心）

在任意页面的 `.js` 顶部：

```javascript
const { api } = require('../../utils/api.js');
const { askCoze } = require('../../utils/coze.js');
const { getLocalUser, needOnboarding } = require('../../utils/auth.js');
```

### 1. 登录后获取用户

```javascript
// 登录成功后在 auth.js 里已写入 globalData
const user = getLocalUser();
// user: { nickName, direction, targetSchool, ... }
```

### 2. 更新备考画像

```javascript
await api('updateProfile', {
  direction: '英语一',
  targetSchool: '中山大学'
});
```

### 3. 保存番茄专注记录

```javascript
await api('addFocusSession', {
  durationMinutes: 25,
  subject: '英语'
});
```

### 4. 获取专注列表（复盘页）

```javascript
const { list } = await api('listFocusSessions', { limit: 20 });
// list[].durationMinutes, date, subject, createdAt
```

### 5. 近 7 日统计（复盘图表）

```javascript
const stats = await api('getWeeklyStats', { days: 7 });
// stats.totalMinutes, stats.sessionCount, stats.byDate['2026-05-21']
```

### 6. AI 答疑

```javascript
const res = await askCoze('考研阅读怎么提高', conversationId);
// res.content, res.conversationId

// 可选：保存记录
await api('addQaLog', { question: '...', answer: res.content });
```

### 7. 是否云开发模式

```javascript
const { useCloud } = require('../../utils/api.js');
// useCloud() === true 表示走云函数；false 为本地演示
```

## 四、本地运行（无需云）

1. 开发者工具 **详情 → 本地设置** → 勾选 **不校验合法域名**  
2. 打开项目后进入登录页，点 **「本地体验（无云开发）」**  
3. 即可调试页面与接口

---

## 五、云开发联调（负责人配置后）

1. `miniprogram/app.js` 里 `env: 'cloud1-xxx'` 已填写  
2. 云控制台集合：`users`、`focus_sessions`、`qa_logs`  
3. 部署云函数：右键 `cloudfunctions/backendApi` → 上传并部署

## 六、你不要改的文件（除非和负责人商量）

- `miniprogram/utils/api.js`
- `miniprogram/utils/coze.js`
- `miniprogram/utils/storage.js`
- `miniprogram/utils/auth.js`
